# Decibells
*Les Sources Notent les Sources. La Vérité Émerge.*

---

## C'est Quoi Une Source

**Tout contenu qui fait des affirmations :**
- Articles de recherche
- Articles de presse  
- Livres
- Rapports
- Posts de blog
- Vidéos
- Podcasts
- Jeux de données

**Chaque source a un coefficient de fiabilité (0.0 à 1.0)**

---

## À Quoi Ressemble L'App

### Écran Principal : Explorateur de Sujets
```
🔍 Recherche : "Changement Climatique"

📊 Sources classées par fiabilité :

🟢 0.94 | Étude Nature Climat (2024)
🟢 0.87 | Rapport GIEC Chapitre 3  
🟡 0.65 | Article BBC Climat
🟡 0.52 | Tribune Guardian
🔴 0.23 | Post Blog Climat
🔴 0.12 | Vidéo YouTube "Vérité sur le Climat"
```

### Page Détail Source
```
📄 "Étude Nature Climat (2024)"
🎯 Fiabilité : 0.94
📈 Notée par : 47 autres sources

Notes Récentes :
🟢 Journal Science (+0.91) ✅ "Méthodologie solide"
🟢 Rapport MIT (+0.88) ✅ "Confirme nos résultats"  
🟡 Site Actu (+0.67) ✅ "Bon résumé"

[Noter Cette Source] (seulement si vous êtes une source)
```

### Interface de Notation
```
Noter : "Article BBC Climat"

Votre source : Étude Nature Climat (poids : 0.94)
Votre note : ⭐⭐⭐⭐⭐ (0.8/1.0)
Commentaire : "Reporting précis, simplifications mineures"

[Envoyer Note]
```

---

## La Magie

**Sources haute fiabilité** → Fort pouvoir de vote → Trouvent d'autres sources fiables
**Sources faible fiabilité** → Votes faibles → Filtrées naturellement

**Résultat :** Les sources cherchant la vérité remontent automatiquement.

---

## Fonctionnalités

✅ **Chercher n'importe quel sujet** - voir sources classées par fiabilité réseau  
✅ **Profils sources** - historique + qui les note  
✅ **Noter autres sources** - si vous êtes vous-même une source  
✅ **Visualisation réseau** - voir comment sources se connectent  
✅ **Sujets tendance** - ce dont parlent les sources maintenant

---

## Pourquoi Ça Marche

**Alignement d'incitations :** Être précis donne plus de pouvoir  
**Effets de réseau :** Sources fiables se regroupent  
**Auto-correction :** Pas besoin de modérateurs  
**Résistance manipulation :** Difficile de feindre fiabilité long-terme
